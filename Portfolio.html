RealSim Inserts

Implemented an automated data simulation script that generates and inserts realistic transactional records into product_orders and order_details tables on a weekly basis. This project emulates a real-world OLTP (Online Transaction Processing) environment to facilitate hands-on practice with time-based data ingestion, analysis, and pipeline design. Also Integrated with ConnectionVault, a custom-built utility that securely manages and abstracts database connection credentials using YAML configuration and runtime selection, eliminating the need for hardcoded connections. Uses Python, MySQL, YAML, SQLAlchemy (Github Repository)
